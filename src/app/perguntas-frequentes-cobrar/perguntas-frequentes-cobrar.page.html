Como posso gerar uma cobrança no Estácio Bank?
Para gerar uma cobrança, acesse a Tela de Cobranças no aplicativo, escolha a opção "Gerar Cobrança" e preencha as informações necessárias, incluindo o valor e a descrição da cobrança.
Quais informações são necessárias para gerar uma cobrança?
Ao gerar uma cobrança, você precisará especificar o valor, a descrição e, opcionalmente, adicionar uma data de vencimento. Certifique-se de incluir os detalhes necessários para uma cobrança precisa.
É possível programar cobranças para uma data futura?
Sim, você pode agendar cobranças para uma data futura. Ao gerar uma cobrança, selecione a opção de agendamento e escolha a data desejada para o vencimento.
Como faço para visualizar as cobranças pendentes na minha conta?
Na Tela de Cobranças, vá para a seção "Cobranças Pendentes" para visualizar todas as cobranças agendadas e seus respectivos status.
Posso editar uma cobrança já gerada?
Não, após criada, não é possível editar uma cobrança.
Existe algum custo associado à geração de cobranças no Estácio Bank?
O Estácio Bank não cobra taxas pela geração de cobranças. Esse serviço é oferecido gratuitamente aos nossos clientes.
Como faço para pagar uma cobrança recebida no Estácio Bank?
Ao receber uma cobrança, você será notificado na Tela de Cobranças. Selecione a cobrança e escolha a opção "Pagar" para efetuar o pagamento.
Qual é o prazo para o pagamento de uma cobrança no Estácio Bank?
O prazo de pagamento de uma cobrança varia de acordo com a data de vencimento especificada durante a geração. Certifique-se de efetuar o pagamento antes do vencimento para evitar atrasos.
Posso visualizar o histórico de cobranças na minha conta?
Sim, você pode acessar o histórico completo de cobranças na Tela de Cobranças para consultar todas as transações passadas.
