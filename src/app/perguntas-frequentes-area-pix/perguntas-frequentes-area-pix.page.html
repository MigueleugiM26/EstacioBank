Como posso cadastrar uma chave Pix no Estácio Bank?
Todos os usuários recebem uma Chave Pix ao criar uma conta no Estácio Bank. Também é possível usar o CPF como Chave Pix.
É possível ter mais de uma chave Pix na mesma conta?
Sim, você pode ter 2 chaves Pix na sua conta do Estácio Bank: seu CPF e uma Chave Aleatória. Assim é possível ter uma maior flexibilidade para receber pagamentos.
Como faço para realizar um pagamento usando a funcionalidade Pix?
Para efetuar um pagamento Pix, vá para a opção "Transferências" na Área Pix, escolha a chave do destinatário e insira o valor desejado. Confirme a transação e pronto!
Existe algum custo associado às transações Pix no Estácio Bank?
O Estácio Bank não cobra taxas por transações Pix. As operações Pix são isentas de tarifas, proporcionando conveniência e economia aos nossos clientes.
Posso agendar pagamentos utilizando o Pix no Estácio Bank?
Sim, é possível agendar pagamentos Pix. Ao realizar uma transferência, escolha a opção de agendamento e selecione a data desejada para a transação.
Como faço para consultar as transações Pix realizadas na minha conta?
No Menu Principal, vá para o histórico de transações para visualizar todas as operações Pix realizadas, incluindo pagamentos e recebimentos.
Existe um limite de valor para transações Pix no Estácio Bank?
O Estácio Bank estabelece limites de valor para transações Pix, que podem variar com base no perfil e histórico do cliente. Consulte a Área Pix para mais informações.
Como posso cancelar uma transação Pix no Estácio Bank?
Em situações específicas, é possível solicitar o cancelamento de uma transação Pix. Acesse a Área Pix, vá para o histórico de transações e siga as instruções para cancelamento, se disponível.
As chaves Pix são sensíveis a maiúsculas e minúsculas?
Sim, as Chaves Aleatórias são sensíveis a maiúsculas e minúsculas. Certifique-se de que você digitou a Chave correta. Informações sobre o destinatário serão exibidas antes da confirmação da transação.
